Open Source Toolkit

Open Source 开源项目

围绕时间序列研究流程沉淀可复用工具，从实验构建、标准评测到智能体编排， 支持更高效、可复现、可扩展的时序研究与应用落地。

面向时序研究的开源基础设施

Open Source 页面汇总课题组在时间序列方向持续建设的工程资产。CastFactory 关注研究与实验的组件化生产， CastEval 关注模型、数据集与应用场景的统一评测，CastAgent 关注将工具调用、实验编排和经验沉淀组织为可扩展的智能体工作流。

Experiment Factory

CastFactory

面向时间序列研究的实验工厂，沉淀数据处理、特征构造、模型配方、训练运行与结果归档等模块， 让新任务能够沿着统一协议快速搭建、复用和复现实验流程。

  • 数据接入、清洗、切分与场景配置模板
  • 模型配方、训练脚本和实验运行规范
  • 面向论文与应用研究的可复现结果资产
Evaluation Suite

CastEval

面向时序模型与智能体系统的统一评测套件，覆盖 benchmark 管理、指标计算、误差归因和报告生成， 帮助研究者在一致口径下比较模型能力与系统收益。

  • 预测、分类、异常检测等任务的评测协议
  • 跨数据集、跨场景、跨模型的指标对齐
  • 面向 ablation、diagnosis 和 leaderboard 的报告结构
Agent Workflow

CastAgent

面向时间序列研究工作流的智能体框架，把任务理解、工具调用、实验编排、结果分析与人工确认串成闭环， 支持从一次性实验走向持续演化的研究自动化。

  • Planner / Executor / Critic 式流程组织
  • 工具增强的数据分析、建模与评测决策
  • 研究记忆、经验复用与人机协同确认机制
