研究组在时间序列分析方向的最新进展与成果发布。

Forecasting · RL 2025

Cast-R1 — 工具增强强化学习驱动的时间序列预测

提出将 Agentic RL 引入时序预测任务：模型通过动态调用数据分析、特征工程等工具，自主完成复杂场景下的多步预测，在 ETTh/ETTm 等标准基准上取得显著提升。

Anomaly Detection 2025

AnomaMind — 面向工业时序异常检测的 Agent 框架

面向真实工业场景，AnomaMind 通过多粒度模式感知与上下文推理，显著降低误报率，支持无监督与少样本两种检测模式。

Foundation Model 2024–2025

InstructTime — 指令驱动的通用时序基础模型

通过大规模多域预训练与指令微调，InstructTime 在 zero-shot 时序预测、分类与插补任务上实现跨领域泛化，无需针对特定数据集重训练。

应用系统

面向真实时序预测任务的可交互 Agent 产品与原型系统。

CastClaw

面向时间序列预测研究的人机协同智能体框架。CastClaw 通过 Planner、Forecaster、Critic 三个专属智能体编排完整预测流程，并在关键节点引入人类确认，将数据分析、特征工程与经典时序模型能力封装为可扩展的运行时工具箱。

📈 多智能体协同 · 人机高效协作 · Agentic Workflow
🧰 运行时工具箱：实验执行沙盒 · 数据分析工具 · 特征分析 · 经典时序模型（ARIMA / Prophet / DLinear / PatchTST …）
